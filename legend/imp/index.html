🧭 Using autosh.php with Proxy and Captcha

With your proxy: add ?proxy=scheme://ip:port (supports ip:port:user:pass)

autosh.php?cc=4111111111111111|12|2025|123&site=https://example.myshopify.com&proxy=http://200.218.236.2:8080

Without proxy: omit ?proxy and a working HTTPS-capable proxy will be auto-picked from ProxyList.txt. If none are available, the script stops with a clear error—fetch proxies first.

autosh.php?cc=4111111111111111|12|2025|123&site=https://example.myshopify.com

Captcha handling: If a captcha appears on products/cart, the script attempts a header-based bypass automatically, reusing your proxy and the current cookie jar.

Runtime tuning: you can adjust speed/timeouts per request:

autosh.php?...&sleep=1&cto=5&to=30&v4=1

sleep: seconds between steps (default 2) · cto: connect timeout seconds (default 8) · to: total timeout seconds (default 45) · v4=1: prefer IPv4
